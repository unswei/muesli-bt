# `integer`

**Signature:** `64-bit integer literal`

## What It Does

Exact integer numeric value.

## Arguments And Return

- Return: integer value

## Errors And Edge Cases

- Overflow in int arithmetic raises errors.

## Examples

### Minimal

```lisp
42
```

### Realistic

```lisp
(* 6 7)
```

## Notes

- Stored as signed 64-bit.
