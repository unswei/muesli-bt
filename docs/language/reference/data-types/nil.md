# `nil`

**Signature:** `literal `nil``

## What It Does

Represents absence of value; falsey in condition position.

## Arguments And Return

- Return: `nil` value

## Errors And Edge Cases

- Self-evaluating.

## Examples

### Minimal

```lisp
nil
```

### Realistic

```lisp
(if nil "then" "else")
```

## Notes

- Common default sentinel.
