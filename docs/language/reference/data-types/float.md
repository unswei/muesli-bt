# `float`

**Signature:** `double literal`

## What It Does

Floating-point numeric value.

## Arguments And Return

- Return: float value

## Errors And Edge Cases

- `write` rejects non-finite floats.

## Examples

### Minimal

```lisp
3.14
```

### Realistic

```lisp
(/ 1 3)
```

## Notes

- Mixed math promotes to float.
