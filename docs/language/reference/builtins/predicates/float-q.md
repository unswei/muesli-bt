# `float?`

**Signature:** `(float? x) -> bool`

## What It Does

Checks whether value is float.

## Arguments And Return

- Arguments: one value
- Return: boolean

## Errors And Edge Cases

- Arity validation errors.

## Examples

### Minimal

```lisp
(float? 3.0)
```

### Realistic

```lisp
(float? 3)
```

## Notes

- Distinguishes float from integer values.
