# `int?`

**Signature:** `(int? x) -> bool`

## What It Does

Alias of `integer?`.

## Arguments And Return

- Arguments: one value
- Return: boolean

## Errors And Edge Cases

- Arity validation errors.

## Examples

### Minimal

```lisp
(int? 3)
```

### Realistic

```lisp
(int? 3.0)
```

## Notes

- Short alias.
