# `log`

**Signature:** `(log x) -> float`

## What It Does

Computes natural logarithm.

## Arguments And Return

- Arguments: one number
- Return: float

## Errors And Edge Cases

- Domain error for `x <= 0`.

## Examples

### Minimal

```lisp
(log 1)
```

### Realistic

```lisp
(log 2.718281828)
```

## Notes

- Natural log base `e`.
