# `/`

**Signature:** `(/ x [y...]) -> float`

## What It Does

Divides numbers.

## Arguments And Return

- Arguments: one or more numbers
- Return: float

## Errors And Edge Cases

- Type errors on non-number.

## Examples

### Minimal

```lisp
(/ 20 2 5)
```

### Realistic

```lisp
(/ 4)
```

## Notes

- Always returns float.
