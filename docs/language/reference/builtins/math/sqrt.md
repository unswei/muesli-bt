# `sqrt`

**Signature:** `(sqrt x) -> float`

## What It Does

Computes square root.

## Arguments And Return

- Arguments: one number
- Return: float

## Errors And Edge Cases

- Domain error for `x < 0`.

## Examples

### Minimal

```lisp
(sqrt 4)
```

### Realistic

```lisp
(sqrt 2.25)
```

## Notes

- Accepts int or float input.
