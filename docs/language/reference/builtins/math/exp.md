# `exp`

**Signature:** `(exp x) -> float`

## What It Does

Computes exponential `e^x`.

## Arguments And Return

- Arguments: one number
- Return: float

## Errors And Edge Cases

- Type/arity validation errors.

## Examples

### Minimal

```lisp
(exp 0)
```

### Realistic

```lisp
(exp 1)
```

## Notes

- Companion to `log`.
